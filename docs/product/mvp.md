# Flaverso - MVP Scope

## Purpose

The purpose of the MVP (Minimum Viable Product) is to validate the core hypothesis of Flaverso:

> Readers are more likely to maintain a reading habit when reading, organization, and progression systems are combined into a single experience.

The MVP focuses on delivering the smallest possible version of the product that provides real value to users.

---

# MVP Objectives

Users should be able to:

- Create an account
- Subscribe to the platform
- Browse available books
- Read ebooks
- Save reading progress
- Organize books in a personal library
- Track reading activity
- Earn XP and achievements

# MVP User Journey

## New User

1. Sign Up
2. Choose reading preferences
3. Browse catalog
4. Add a book to library
5. Start reading
6. Earn XP
7. Unlock first achievement

---

# MVP Success Criteria

The MVP will be considered successful if users:

- Read consistently
- Return to the platform regularly
- Complete books
- Maintain reading streaks
- Engage with progression systems

# MVP Principle

When in doubt:

Prioritize reading.

If a feature does not directly improve reading, organization, discovery, or progression, it should be postponed to a future release.
